### How to use:

server:

- cd to `server`, npm install
- `npm start` or `sudo node server.js`
- server will run on `PORT: 80`

client:

- cd to `client`, `npm install` or `yarn`
- open file `./client/src/api/index.js`
  - update PORT to 80
- `npm start`
